# Tracker

> **A lightweight, Rust-based tracking utility**

Tracker provides a high-performance system for monitoring and tracking specific events or metrics.

---

## 🚀 Features

- **Safe and Fast**: Written in Rust for maximum performance and memory safety.
- **Configurable**: Driven by `config.default.toml` to adapt to various tracking needs.
- **Windows Integration**: Includes Windows resource files (`app.rc`, `app.ico`) for a polished desktop experience.

## 🛠️ Tech Stack

- **Rust**: Core application.
- **TOML**: Configuration syntax.

## 📦 Getting Started

### Prerequisites

- Rust toolchain (`cargo`)

### Setup and Build

1. Copy `config.default.toml` to a local `config.toml` and adjust your settings.
2. Build the application:
   ```bash
   cargo build --release
   ```

*Check `Instruction.md` for advanced usage and detailed tracking documentation.*
