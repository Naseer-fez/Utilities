# QuickFinder

> **Instant desktop search utility**

QuickFinder is a fast, lightweight file lookup tool with automated asset handling. 

---

## 🚀 Features

- **Asset Management**: Includes `convert_assets.py` to seamlessly handle application resources.
- **Automated Builds**: Simple `build.bat` workflow to compile and set up the tool.

## 📦 Getting Started

### Building the Project

Run the build script to compile the application and convert necessary assets:

```cmd
build.bat
```

The compiled output will be available in the `bin/` directory.
