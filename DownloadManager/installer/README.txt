Download Manager Setup
======================

For normal users
----------------
1. Double-click the file named DownloadManagerSetup-1.1.0-win64.msi.
2. Keep the default install location unless you have a reason to change it.
3. Leave "Launch Download Manager" checked at the end of the installer.
4. When the app opens, follow the Setup Wizard.

After installation
------------------
- Open it from the Start Menu by searching for "Download Manager".
- A desktop shortcut is also created.
- Your settings are stored in your Windows user profile, not in Program Files.
- You can run the Setup Wizard again from Settings.

Uninstall
---------
Use Windows Settings > Apps > Installed apps > Download Manager > Uninstall.

Important
---------
Use this app only for media you own, created yourself, or are otherwise allowed
to download.
