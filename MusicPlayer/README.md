# MusicPlayer

> **A custom local audio player with a dedicated installer**

MusicPlayer is a desktop utility for playing local audio files. It features a custom user interface and comes with an automated Nullsoft Scriptable Install System (NSIS) installer.

---

## 🚀 Features

- **Local Audio Playback**: Reliable and lightweight music playing functionality.
- **Standalone Installer**: Bundled as `AntigravityPlayerSetup.exe` built from `installer.nsi`.
- **Easy Distribution**: Pre-compiled binary (`player.exe`) for straightforward deployment.

## 🛠️ Tech Stack

- **NSIS**: For building the Windows installer.
- **Batch Scripts**: Build automation (`build.bat`).

## 📦 Getting Started

### Installation

1. Run the `AntigravityPlayerSetup.exe` to install the player.
2. Launch the application from your desktop or start menu.

### Building from Source

To recompile or package the application, run:
```cmd
build.bat
```
