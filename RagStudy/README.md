# RagStudy

> **AI-Powered Retrieval-Augmented Generation for Study Notes**

RagStudy is a full-stack application leveraging RAG (Retrieval-Augmented Generation) to help users query, study, and interact with their document knowledge base effectively.

---

## 🚀 Features

- **RAG Architecture**: Combines language models with local data retrieval to ensure accurate, context-aware answers.
- **Full-Stack Application**: Separated frontend and backend layers for a robust architecture.
- **Easy Bootstrapping**: One-click startup using Windows batch scripts.

## 🛠️ Tech Stack

- **Python**: Backend server logic.
- **Web Technologies**: Frontend interface.

## 📦 Getting Started

### Prerequisites

- Python 3.10+
- Install dependencies:
  ```bash
  pip install -r requirements.txt
  ```

### Configuration & Run

1. Configure your environment variables in the `.env` file.
2. Launch the application stack:
   ```cmd
   start.bat
   ```
   *Alternatively, run the python entry point directly:*
   ```bash
   python run.py
   ```
